学习Python3爬虫（一）爬取贴吧用户头像

最近在看O'Reilly Media的Web Scraping with Python这本书，第一次看英文原书但是看起来并不是很困难。应该是一本不错的爬虫入门书。我会把自己写的一些小爬虫放到我的github上，欢迎围观orz

获取网页HTML数据

Python3提供了一个urlopen函数（在urllib.request中），可以用来打开url并获取url的HTML数据。

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
from urllib.request import urlopen
url = r'http://cn.bing.com'
html = urlopen(url)
print(html.read())

上面这段代码的功能即为打印必应首页的HTML数据

urlopen()还有其他可选参数，如context参数可以指定访问https时使用的协议版本等，具体见官方文档

解析HTML

得到HTML数据后，接下来的问题就是如何获取我们所需要爬取的信息。这时就需要解析HTML了。书中使用的是BeautifulSoup。

pip install BeautifulSoup4

同时还需安装lxml解析器

pip install lxml

当然也可以使用别的解析器，具体区别见这里

其使用具体见文档

解析举例：若现有一个html，要解析出所有的Tag为a，class属性为userinfo_head的数据，则可这样写：

html = urlopen(url)
bs = BeautifulSoup(html.read(), 'lxml')
pics = bs.findAll('a', {'class':'userinfo_head'})

应用举例

这个网址是百度WP7吧的经验值排行页面，现在想要爬取排行榜前n页的吧友的头像。

首先分析排行榜的url，可以发现其是http://tieba.baidu.com/f/like/furank?kw=前缀+wp7吧名+&pn=+1页数组成的。于是可以很容易的构造前n页的url

url = r'http://tieba.baidu.com/f/like/furank?kw=' + tieba_name + '&pn=';
for i in range(1, pages+1):
    get_usr_url(url+str(i))

利用chrome的审查元素功能，可以发现排行榜前3的用户在class属性为drl_item_name_top的Taga中，且其href属性对应了其个人贴吧主页地址；而其余用户的区别在于class属性为drl_item_name_nor 所以对于排行榜的某页的url，可以这样获取该页上所有用户的主页地址

html = urlopen(url)
bs = BeautifulSoup(html.read(), 'lxml')
usrs = bs.findAll('a', {'class':'drl_item_name_top'}) + bs.findAll('a', {'class':'drl_item_name_nor'})
for usr in usrs:
    download_usr_pic(r'http://tieba.baidu.com'+usr['href'], usr.get_text())

同理可解析出个人主页上的头像url包含在bs.findAll('a', {'class':'userinfo_head'})的src属性中,但该url指向的图片是经过压缩的，并不是我们想要的原图。对比原图的url和压缩后图片的url可以发现，两个url最后一个斜杠后的那一串是一样的，于是可以由压缩过的图片的url来获得原图的url

html = urlopen(url)
bs = BeautifulSoup(html.read(), 'lxml')
pics = bs.findAll('a', {'class':'userinfo_head'})[0]
for pic in pics:
    pic_url = r'http://himg.baidu.com/sys/portraitl/item/'+pic['src'].split('/')[-1].split('?')[0]

完整的实现代码：

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
'''
百度贴吧用户头像原图爬虫（以该吧经验值从高到低开始爬）
'''
from urllib.request import urlopen, Request
from bs4 import BeautifulSoup
import os
def download_usr_pic(url, name):#获取并下载用户头像
    html = urlopen(url)
    bs = BeautifulSoup(html.read(), 'lxml')
    pics = bs.findAll('a', {'class':'userinfo_head'})[0]
    for pic in pics:
        pic_url = r'http://himg.baidu.com/sys/portraitl/item/'+pic['src'].split('/')[-1].split('?')[0]#头像原图地址
        with open(name+'.jpg', 'wb') as fout:
            fout.write(urlopen(pic_url).read())
            print(name, '....ok')

def get_usr_url(url):#获取url页面上所有用户的个人主页地址
    html = urlopen(url)
    bs = BeautifulSoup(html.read(), 'lxml')
    usrs = bs.findAll('a', {'class':'drl_item_name_top'}) + bs.findAll('a', {'class':'drl_item_name_nor'})
    for usr in usrs:
        download_usr_pic(r'http://tieba.baidu.com'+usr['href'], usr.get_text())

dir = 'tieba_portrait'
try:
    os.mkdir(dir)
except:
    pass
os.chdir(dir)
tieba_name = input('Please input the name of tieba...\n')
tieba_name = str(tieba_name.encode('GB2312')).replace(r'\x','%').split("'")[1]#进行网址的GB2312编码
pages = int(input('How many pages do you want?\n'))
url = r'http://tieba.baidu.com/f/like/furank?kw=' + tieba_name + '&pn=';
for i in range(1, pages+1):
    get_usr_url(url+str(i))
